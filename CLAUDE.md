# ShowPur Development Rules

## Product Vision
ShowPur is a simple SME Business OS for Bangladesh.

Core Principles:
- Simplicity first
- Cashflow first
- Mobile first
- SME owner focused
- Avoid ERP complexity

## Backend Rules
- Use Django REST Framework
- Use BusinessProfile as the primary business entity
- Keep models normalized
- Use ViewSets and Serializers

## Frontend Rules
- React functional components
- Mobile-first UI
- Large action buttons
- Simple forms

## AcShow Rules
Money In:
- Sales
- Collections
- Other Income

Money Out:
- Purchases
- Supplier Payments
- Expenses

Inventory:
- Purchase increases stock
- Sale decreases stock

Always prioritize SME usability over accounting complexity.
